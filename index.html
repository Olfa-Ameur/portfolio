About Me

Embedded Systems Engineer

Hello! I'm Olfa, an Embedded Systems and IoT Engineer passionate about designing intelligent connected devices that bridge hardware and software. My work focuses on developing real-world embedded systems using microcontrollers such as STM32 and ESP32, integrating wireless communication technologies like Bluetooth Low Energy, sensor data processing, and IoT architectures. I enjoy building complete systems—from low-level firmware and hardware integration to connected applications. My projects include BLE-based driver identification systems for fleet management, IoT smart devices with computer vision capabilities, and embedded health monitoring solutions. Through these projects, I aim to create reliable, efficient, and innovative embedded solutions that solve practical problems in areas such as smart devices, connected vehicles, and intelligent monitoring systems.

My Projects

BLE Health Monitoring Node

Embedded health monitoring system built with ESP32-C3 using Bluetooth Low Energy. The device measures heart rate using a sensor, displays the data on an LCD screen, and transmits physiological data via BLE. BLE packets were analyzed using Wireshark and an nRF51 sniffer for protocol inspection.

BLE Fleet Management System

Embedded fleet management authentication system that replaces traditional iButton driver identification with secure Bluetooth Low Energy communication. The solution uses an STM32WB55 BLE node that communicates with a tracking unit via UART, while GPS data is transmitted through the fleet management device.

Smart Fridge IoT System
